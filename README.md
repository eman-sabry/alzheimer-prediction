# Alzheimer Disease Prediction 

A web application built with **Flask + Deep Learning** to predict the stage of Alzheimer disease from MRI images.

## Features
- Upload MRI scan
- Preprocessing using OpenCV
- Deep Learning model (.keras)
- Instant prediction
- Beautiful dark-themed UI

##  Tech Stack
- Python (Flask)
- TensorFlow / Keras
- HTML, CSS
- OpenCV
- Vercel Deployment (frontend)
